# Best Practices Guide

This guide covers best practices for developing, testing, and maintaining MCP servers using this template.

## Code Organization

### Project Structure

Follow the modular structure provided by the template:

```
src/
├── models/              # Domain models and DTOs
├── mcp/                 # MCP server implementation
│   ├── tools/          # Tool implementations
│   └── models/         # MCP-specific models
├── web_server/         # FastAPI web server
│   ├── routers/        # API route handlers
│   └── models/         # Request/response models
└── integrate/          # Integration layer
```

### Module Responsibilities

- **models/**: Define data structures and validation
- **mcp/**: Implement MCP tools and server logic
- **web_server/**: Define API endpoints and web logic
- **integrate/**: Combine MCP and web server

### Naming Conventions

```python
# Functions and variables: snake_case

    pass

# Models: Descriptive names with suffix

    pass

    pass
```

## Type Safety

### Use Type Hints

Always use type hints for functions and variables:

```python
from typing import Optional, List, Dict

def process_data(
    items: List[str],
    config: Optional[Dict[str, str]] = None
) -> Dict[str, int]:
    """Process data and return results."""
    result: Dict[str, int] = {}
    for item in items:
        result[item] = len(item)
    return result
```

### Pydantic Models

Use Pydantic models for all data structures:

```python
from pydantic import BaseModel, Field, field_validator

class UserData(BaseModel):
    """User data model with validation."""
    
    name: str = Field(
        min_length=1,
        max_length=100,
        description="User name"
    )
    email: str = Field(
        pattern=r"^[^@]+@[^@]+\.[^@]+$",
        description="User email"
    )
    age: int = Field(
        ge=0,
        le=150,
        description="User age"
    )
    
    @field_validator('name')
    @classmethod
    def validate_name(cls, v: str) -> str:
        """Validate name format."""
        if not v.strip():
            raise ValueError('Name cannot be empty')
        return v.strip()
```

## Error Handling

### Graceful Error Handling

Always handle errors gracefully:

```python
from typing import Optional

class ToolOutput(BaseModel):
    """Output with error handling."""
    success: bool = Field(description="Whether operation succeeded")
    data: Optional[dict] = Field(default=None, description="Result data")
    error: Optional[str] = Field(default=None, description="Error message")
    error_code: Optional[str] = Field(default=None, description="Error code")

@mcp_factory.get().tool()
def my_tool(input: MyInput) -> ToolOutput:
    """Tool with comprehensive error handling."""
    try:
        result = perform_operation(input)
        return ToolOutput(success=True, data=result)
    except ValueError as e:
        return ToolOutput(
            success=False,
            error=str(e),
            error_code="VALIDATION_ERROR"
        )
    except Exception as e:
        return ToolOutput(
            success=False,
            error=f"Unexpected error: {str(e)}",
            error_code="INTERNAL_ERROR"
        )
```

### Logging Errors

Log errors with context:

```python
import logging

logger = logging.getLogger(__name__)

@mcp_factory.get().tool()
def my_tool(input: MyInput) -> ToolOutput:
    """Tool with logging."""
    try:
        logger.info(f"Processing tool with input: {input}")
        result = perform_operation(input)
        logger.info(f"Tool completed successfully")
        return ToolOutput(success=True, data=result)
    except Exception as e:
        logger.error(
            f"Tool failed with error: {str(e)}",
            exc_info=True,
            extra={"input": input.dict()}
        )
        return ToolOutput(success=False, error=str(e))
```

### Unit Testing

Write focused unit tests:

```python
import pytest
from src.mcp.models.input.calculator_input import CalculatorInput
from src.mcp.tools.calculator import calculate

class TestCalculator:
    """Test calculator tool."""
    
    def test_simple_addition(self):
        """Test simple addition."""
        input_data = CalculatorInput(expression="2 + 2")
        output = calculate(input_data)
        
        assert output.result == 4.0
        assert output.error is None
    
    def test_division_by_zero(self):
        """Test division by zero error handling."""
        input_data = CalculatorInput(expression="1 / 0")
        output = calculate(input_data)
        
        assert output.success is False
        assert output.error is not None
    
    @pytest.mark.parametrize("expression,expected", [
        ("2 + 2", 4.0),
        ("10 - 3", 7.0),
        ("3 * 4", 12.0),
        ("10 / 2", 5.0),
    ])
    def test_arithmetic_operations(self, expression, expected):
        """Test various arithmetic operations."""
        input_data = CalculatorInput(expression=expression)
        output = calculate(input_data)
        
        assert output.result == expected
```

```python
from cachetools import TTLCache
import time
```

### Async Operations

Use async for I/O-bound operations:

```python
import asyncio
from fastapi import APIRouter

router = APIRouter()

async def fetch_data_1():
    """Simulate async I/O."""
    await asyncio.sleep(1)
    return "data1"

async def fetch_data_2():
    """Simulate async I/O."""
    await asyncio.sleep(1)
```

## Security

### Input Validation

Always validate input:

```python
from pydantic import BaseModel, Field, validator

class SecureInput(BaseModel):
    """Input with security validation."""
    
    username: str = Field(
        min_length=3,
        max_length=50,
        pattern=r"^[a-zA-Z0-9_]+$",
        description="Username (alphanumeric and underscore only)"
    )
    password: str = Field(
        min_length=8,
        description="Password (minimum 8 characters)"
    )
    
    @validator('password')
    def validate_password_strength(cls, v):
        """Validate password strength."""
        if not any(c.isupper() for c in v):
            raise ValueError('Password must contain uppercase letter')
        if not any(c.isdigit() for c in v):
            raise ValueError('Password must contain digit')
        if not any(c in '!@#$%^&*' for c in v):
            raise ValueError('Password must contain special character')
        return v
```
