# MCP Server Development Requirements

Before you begin contributing to the MCP Server Template, ensure your development environment meets these requirements.

## Software Prerequisites

### Required Software

- **Python**: Python 3.12 or higher (3.13 recommended)
- **Git**: Latest stable version
- **uv**: Latest version (critical for this project)

### Recommended Development Tools

- **Visual Studio Code**, **PyCharm**, or other IDE with Python support
- **pre-commit**: For running pre-commit hooks (included in template)
- **Docker**: For containerized testing and deployment (optional)

## MCP Server Dependencies

The MCP server template includes these key dependencies that will be installed automatically:

### Core MCP Server Stack
- **FastAPI**: Web framework for API endpoints and MCP server
- **Pydantic**: Data validation and settings management
- **mcp**: Model Context Protocol implementation
- **uvicorn**: ASGI server for running the application

### Development Tools
- **pytest**: Testing framework with async support
- **pytest-cov**: Coverage reporting
- **pytest-asyncio**: Async testing support
- **black**: Code formatting
- **ruff**: Fast Python linting
- **mypy**: Static type checking
- **pre-commit**: Pre-commit hooks management

## Development Environment Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Chisanan232/Template-Python-Base-MCP-Server.git
   cd Template-Python-Base-MCP-Server
   ```

2. **Install uv Package Manager** (if not already installed):
   
   <Tabs>
     <TabItem value="macos" label="macOS">
     ```bash
     # Using Homebrew
     brew install uv
     
     # Or using curl
     curl -LsSf https://astral.sh/uv/install.sh | sh
     ```

     </TabItem>
     <TabItem value="linux" label="Linux">
     ```bash
     # Using curl
     curl -LsSf https://astral.sh/uv/install.sh | sh
     
     # Or using pip
     pip install uv
     ```

     </TabItem>
     <TabItem value="windows" label="Windows">
     ```powershell
     # Using PowerShell
     powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
     
     # Or using pip
     pip install uv
     ```

     </TabItem>
   </Tabs>

3. **Install project dependencies**:
   ```bash
   # Install all dependencies including dev tools
   uv sync
   ```

4. **Set up pre-commit hooks**:
   ```bash
   # Install pre-commit hooks for code quality
   uv run pre-commit install
   
   # Test pre-commit setup
   uv run pre-commit run --all-files
   ```

5. **Create a `.env` file for local development**:
   ```bash
   cp .env.example .env
   # Edit .env with your development configuration
   ```

## Next Steps

Once your development environment is set up, refer to the [Development Workflow](./workflow.mdx) documentation for the next steps.
